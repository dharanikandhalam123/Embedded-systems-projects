## Overview
An IoT-based embedded system to monitor water quality parameters and visualize them in real time using a cloud dashboard.

## Hardware Used
- STM32L4S5 IoT Discovery Kit
- pH sensor
- TDS sensor
- Turbidity sensor
- DS18B20 temperature sensor
- ADS1115 ADC module

## Software & Tools
- STM32CubeIDE
- Embedded C
- I2C and ADC interfaces
- ThingSpeak cloud platform

## Description
Sensor data is acquired using the STM32 microcontroller, processed locally, and transmitted to the cloud for real-time monitoring and historical analysis.
