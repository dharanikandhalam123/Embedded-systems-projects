# Embedded Systems Projects

## Technologies & Domains
- Embedded C / C++
- STM32 Microcontrollers
- ARM Architecture & IoT systems and cloud integration
- VLSI circuit design and analysis

- 4x4 DRAM Design (VLSI Project)


Each project folder includes a detailed README describing the system architecture, tools used, and implementation overview.
